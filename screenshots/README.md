## How to Take Screenshots

### From Android Emulator:
```
In Android Studio → Emulator → Camera icon on the right panel
```

### From Physical Device:
```
Volume Down + Power button simultaneously
```

### From Android Studio (Logcat):
```
View → Tool Windows → Logcat → Camera icon
```

## Size Requirements (Play Store)
- Phone screenshots: 1080 x 1920 px (portrait)
- Tablet screenshots: 1920 x 1080 px (landscape)
- Format: PNG or JPEG
